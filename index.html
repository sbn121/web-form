웹 폼태그

입력 폼을 구성하는 태그로, 사용자로부터 어떠한 값을 입력받기 위한 태그들을 가리킵니다.

이러한 태그는 주로 <form>과 <form> 사이에 넣어서 사용자 입력화면을 구현합니다.

웹 폼태그의 종류

  • form : 여기서~여기까지가 폼(=입력받는) 양식이다
  • label : 라벨 (표식)
  • fieldset : 폼의 구성이 복잡할 때 div대신 구역을 구분용
  • input : 한 줄 입력 (ex. 검색어 입력란)
  • select : 선택 옵션 제공(ex. 수량 선택)
  • textarea : 여러줄 입력 (ex. 자기소개)
  • button : 버튼 요소 (ex. 가입 버튼, 취소 버튼, 검색 버튼)
  • datalist : select 처럼 여러 옵션값을 취하는 목록(잘안씀)
  • keygen : 보안키를 만들 때 사용(잘안씀)
  • output : 수학적 계산결과를 출력하는 폼(잘안씀)

각 폼태그 마다 각기 다른 속성과 값을 가질 수 있습니다.
